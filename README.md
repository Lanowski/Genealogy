# Genealogy

Генеалогический исследовательский проект семьи Грязнухиных с интерактивной визуализацией

**🌐 Live Demo:** [Открыть приложение](https://lanowski.github.io/Genealogy)

## Особенности

- 🌳 **Интерактивное семейное дерево** — организовано по поколениям с индикаторами достоверности
- 📅 **Хронология событий** — все подтверждённые и гипотетические события с источниками  
- 📚 **Полная библиография** — 13+ архивных и опубликованных источников
- 🗃 **Архивный план** — приоритизированные запросы с контактами архивов

## Технологический стек

- **React 18** — для интерактивного интерфейса
- **Vite** — быстрая сборка и HMR
- **Georgia serif + кастомная палитра** — исторический стиль

## Запуск локально

```bash
npm install
npm run dev
```

Приложение откроется на http://localhost:3000
